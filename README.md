# Personal Portfolio - Dhruv Chhikara 🚀

A dynamic, responsive personal portfolio website built with Python (Flask), HTML, CSS, and JavaScript. 

## ✨ Features
- **Dynamic Content:** All skills, projects, and certifications are served dynamically from the Python backend.
- **Two-Column Grid:** Clean separation of Projects and Certifications.
- **Interactive UI:** Clickable image modals, smooth hover effects, and integrated FontAwesome icons.
- **Working Contact Form:** Direct email functionality via Formspree.

## 🛠️ Tech Stack
- **Backend:** Python 3, Flask
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Deployment:** Vercel
